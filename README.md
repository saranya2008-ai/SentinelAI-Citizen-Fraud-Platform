# SentinelAI-Citizen-Fraud-Platform

[![Open in Bolt](https://bolt.new/static/open-in-bolt.svg)](https://bolt.new/~/sb1-wk3f3sed)

An AI-powered web application developed for ET AI Hackathon 2.0 to detect digital arrest scams, phishing attacks, government impersonation, banking fraud, and other online scams. SentinelAI helps users identify suspicious messages, verify authorities, assess fraud risk, and receive actionable safety recommendations.

## Tech Stack
- HTML
- CSS
- JavaScript
- Bolt.new
- GitHub
